# 🔍 GitHub User Finder

GitHub REST API kullanarak geliştirilmiş, modern ve kullanıcı odaklı bir portfolyo inceleme uygulamasıdır. Kullanıcıların istatistiklerini ve güncel projelerini şık bir arayüzle sunar.

### 🚀 Canlı Demo
Uygulamayı buradan inceleyebilirsiniz: [https://github-user-finder-green.vercel.app/](https://github-user-finder-green.vercel.app/)

---

## ✨ Öne Çıkan Özellikler

* **Dinamik Tema Sistemi:** Özel belirlenmiş renk paleti ile (Deep Navy & Cloud White) pürüzsüz Dark/Light mode geçişi.
* **Gelişmiş Arama Deneyimi:** Kapsül (pill-shaped) tasarımlı, tek tıkla temizleme özelliğine sahip akıllı arama çubuğu.
* **Asenkron Veri Yönetimi:** `fetch` API ve `async/await` yapısı kullanılarak GitHub profilleri ve son 4 projenin anlık çekilmesi.
* **Tam Responsive Yapı:** Tailwind CSS v4 ile geliştirilen, mobil ve masaüstü cihazlara tam uyumlu esnek grid mimarisi.

## 🛠️ Teknik Yığın (Tech Stack)

* **Frontend:** React 18 (Vite)
* **Dil:** TypeScript
* **Styling:** Tailwind CSS v4
* **İkonlar:** React Icons
* **Deployment:** Vercel

## ⚙️ Yerel Kurulum

Projeyi kendi bilgisayarınızda çalıştırmak için:

1. Depoyu klonlayın: `git clone https://github.com/EsmanKels/github-user-finder.git`
2. Bağımlılıkları yükleyin: `npm install`
3. Uygulamayı başlatın: `npm run dev`
